# Конверт валют

Даннный проект конвертирует любые валюты

## Установка

Используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Пример запуска скрипта

Для запуска скрипта у вас уже должен быть утановлен Python3.
Чтобы запустить программу надо в концоле прописать примерно следующие:
```
python main.py --base kzt --target usd --amount 100000
```

## Переменные окружения

Часть настроек проекта берётся из переменных окружения. Перменные окружения - это переменные, значения которые присваиваются программе Python извне. Чтобы их определить, создайте файл .env рядом с main.py и запишите туда данные в таком формате: ПЕРЕМЕННАЯ=значение.

Пример содержания файла .env:
```
API_KEY = "0f67e867867367067567b67f5267f672675b11b960221"
```
Получить ключь можно на сайте [Exchange Rate API](https://www.exchangerate-api.com/)
