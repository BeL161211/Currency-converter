# Конверт валют

Даннный проект конвертирует любые валюты

## Установка

Используйте pip (или pip3, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```

## Пример запуска скрипта

Для запуска скрипта у вас уже должен быть утановлен Python3.
Чтобы запустить программу надо в концоле прописать примерно следующие:
```
python main.py --base kzt --target usd --amount 100000
```
